# LA NOCHE DE REYES - Estructura 100 Escenas

## Datos de Referencia (análisis real)
- Austin Powers 1: 132 escenas / 137 págs = **0.96 págs/escena**
- Aliens (Cameron): 192 escenas / 158 págs = **0.82 págs/escena**

**Target para comedia rápida de 90 páginas: ~100 escenas (~0.9 págs/escena)**
